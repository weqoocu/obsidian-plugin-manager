# Obsidian Plugin Manager

一个用于管理 Obsidian 插件的实用工具。

## 功能

- **插件列表** - 显示所有已启用的插件，包含名称、版本、ID 和描述
- **搜索过滤** - 支持按插件名称、ID 或描述快速搜索
- **刷新插件** - 重新加载插件（先停用再启用），同时刷新版本信息
- **打开文件夹** - 在系统文件管理器中打开插件目录
- **GitHub 链接** - 快速跳转到插件的 GitHub 仓库
- **打开设置** - 直接跳转到插件的设置页面

## 安装

### 手动安装

1. 下载最新的 release
2. 解压到 Obsidian 插件目录：`<vault>/.obsidian/plugins/obsidian-plugin-manager/`
3. 重启 Obsidian
4. 在设置 → 社区插件中启用 "Plugin Manager"

### BRAT 安装

1. 安装 [BRAT](https://github.com/TfTHacker/obsidian42-brat) 插件
2. 添加 beta 插件：`weqoocu/obsidian-plugin-manager`

## 使用方法

- 点击左侧边栏的拼图图标打开插件管理器
- 或使用命令面板搜索 "Plugin Manager"

### 命令

- `Open Plugin Manager` - 打开插件管理器
- `Refresh all enabled plugins` - 批量刷新所有插件
